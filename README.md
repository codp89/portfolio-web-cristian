## Cómo probar

Abrir `index.html` en el navegador.

## Cambios importantes antes de publicar

1. Reemplazar el número de WhatsApp:
   Buscar `5491100000000` en `index.html` y cambiarlo por tu número real con código de país.

2. Reemplazar links de LinkedIn y GitHub:
   Buscar estos enlaces:
   - `<a href="#" target="_blank">💼 LinkedIn</a>`
   - `<a href="#" target="_blank">🐙 GitHub</a>`

3. Cambiar textos si quieres especializarte en:
   - PHP/Symfony
   - Laravel
   - Python/FastAPI
   - APIs e integraciones
   - Mantenimiento de sistemas

## Publicación gratis recomendada

- Netlify: arrastrar la carpeta o el ZIP.
- GitHub Pages: subir los archivos a un repositorio y activar Pages.
- Vercel: importar desde GitHub.
